# Create applications with the Copilot CLI

## Calculator CLI

A Node.js CLI calculator supporting arithmetic operations.

**Usage:** `node src/calculator.js <operation> <number1> [number2]`

### Supported operations

| Operation | Aliases | Example |
|-----------|---------|---------|
| addition | `add`, `+`, `addition` | `node src/calculator.js add 2 3` |
| subtraction | `subtract`, `-`, `subtraction` | `node src/calculator.js subtract 10 4` |
| multiplication | `multiply`, `*`, `x`, `multiplication` | `node src/calculator.js multiply 45 2` |
| division | `divide`, `/`, `division` | `node src/calculator.js divide 20 5` |
| modulo | `modulo`, `%`, `mod` | `node src/calculator.js modulo 5 2` |
| power | `power`, `^`, `**`, `pow`, `exponentiation` | `node src/calculator.js power 2 3` |
| square root | `sqrt`, `squareroot` | `node src/calculator.js sqrt 16` |

### Error handling

- Division by zero and modulo by zero produce a clear error message.
- Square root of a negative number produces a clear error message.

### Running tests

```bash
npm test
```
